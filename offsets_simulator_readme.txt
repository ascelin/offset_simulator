run_model.R - used to input policy parameters, controls system, and talks to run_system_routines.R collate_routines.R and plot_routines.R
initialise_program_params.R - store policy parameters in list to be used when running 
initialise_common_params.R - common ecology and landscape parameters used to test policies on
run_system_routines.R - functions to run simulation and generate simulation outputs
collate_routines.R - collate system outputs and organise into outputs to plot
plot_routines.R - use collated system outputs and run plot series
